Log Extractor — десктопное приложение для извлечения, фильтрации и анализа логов из файлов в формате JSON Lines. Утилита предоставляет графический интерфейс для работы с логами Directum RX.


ВОЗМОЖНОСТИ
-----------

Основные функции:
  - Извлечение логов из одного или нескольких файлов одновременно
  - Поддержка rotated logs (файлы с индексами: *.0.log, *.1.log, *.log)
  - Автоматическая хронологическая сортировка записей
  - Парсинг JSON-строк с внутренней и внешней временной меткой
  - Цветовая подсветка записей по уровням логов

Фильтрация:
  - По времени: указание начальной и конечной даты и времени
  - По уровням: DEBUG, INFO, WARN, ERROR (выбор чекбоксами)
  - По источнику: фильтр только записей конвертера (Logger)
  - По содержанию: фильтр только ключевых записей конвертации
  - По тексту: поиск по содержанию сообщений

Экспорт данных:
  - Сохранение отфильтрованных записей в новый файл
  - Копирование отдельных строк в буфер обмена
  - Копирование всех отфильтрованных записей

ИСПОЛЬЗОВАНИЕ
-------------

Запуск приложения:
  python main.py

Порядок работы:
  1. Выбор файлов: нажмите кнопку "Обзор..." и выберите один или несколько .log файлов
  2. Диапазон времени: укажите начальную и конечную дату и время (формат: YYYY-MM-DD HH:MM:SS)
  3. Уровни логов: отметьте нужные уровни (DEBUG, INFO, WARN, ERROR)
  4. Дополнительные фильтры:
       - "Только конвертация" — показывает только записи от Logger
       - "Только ключевые записи" — показывает сообщения о запуске/завершении конвертации
  5. Текстовый фильтр: введите подстроку для поиска в сообщениях
  6. Извлечение: нажмите кнопку "Извлечь" для выполнения фильтрации
  7. Сохранение: нажмите "Сохранить" для экспорта результатов в файл

Горячие клавиши и контекстное меню:
  - ПКМ на таблице — контекстное меню с опциями копирования
  - "Копировать строку" — копирует выделенные строки в буфер обмена
  - "Копировать всё" — копирует все отфильтрованные записи

КЛЮЧЕВЫЕ ЗАПИСИ КОНВЕРТАЦИИ
---------------------------

Утилита распознает следующие сообщения как ключевые:
  - "Запущена постановка в очередь конвертации писем в docx"
  - "Найдено писем для постановки в очередь:"
  - "Постановка в очередь завершена. Всего"
  - Сообщения, начинающиеся с "ConvertOnEmail:"
